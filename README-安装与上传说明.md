# 茶样志 PWA V1.2 全意见凝练版

本版本直接基于你上传的完整项目修改，保留原有首页、批次、拍照、样品录入、历史记录、CSV导出和数据备份。

## 本次优化

### 1. 有高频词时

分别汇总所有人的香气意见和滋味意见，统计词频。

如果多个词并列最高频，会显示全部并列最高频词，不只显示第一个。

### 2. 没有高频词时

当所有提示词都只出现一次时，系统会基于全部提示词进行凝练：

- 汇总全部意见
- 同义词统一
- 删除重复词
- 合并被复合词包含的泛化词
- 显示全部凝练后的代表词

例如：

- 蜂蜜香、蜜糖香 → 蜜香
- 桂圆味、龙眼香 → 桂圆香
- 甜润、入口甜 → 甜醇
- 木质香、木头香 → 木香

不会再只显示第一个提示词。

### 3. 颜色

- 绿色：该特征词也出现在“我的香气/滋味”中
- 红色：该特征词未出现在“我的香气/滋味”中

## 更新方法

1. 建议先从旧版导出 JSON 备份。
2. 解压本 ZIP。
3. 将解压后的全部文件和 assets 文件夹上传到原 GitHub 仓库根目录。
4. 覆盖旧文件并点击 Commit changes。
5. 等待 GitHub Pages 部署完成。
6. 在 iPhone Safari 中刷新页面。
7. 若桌面 App 仍显示旧版，完全关闭后重开；必要时删除旧桌面图标，再从 Safari 重新添加到主屏幕。
