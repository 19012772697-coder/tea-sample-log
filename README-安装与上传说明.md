# 茶样志 PWA V1.2 特征点优化版

本版本直接基于你上传的完整项目修改，保留首页、批次、拍照、输入、历史记录、导出与备份。

## 特征点规则

- 香气和滋味分别分析。
- 汇总“我的品评”和“他人的品评”。
- 换行、分号和句号可以分隔多条意见。
- 同一条意见中的同一术语只计一次。
- 有重复词时，显示全部并列最高频词，不限制为第一个或前三个。
- 所有词频都为1时，显示全部去重提示词，不再只显示第一个。
- 与“我的香气/滋味”相同的词显示绿色。
- 未出现在“我的香气/滋味”中的词显示红色。

## 更新方法

1. 建议先在旧版中导出 JSON 备份。
2. 解压 ZIP。
3. 将解压后的全部文件和 assets 文件夹上传至原 GitHub 仓库根目录并覆盖。
4. 点击 Commit changes。
5. 等待 GitHub Pages 部署完成。
6. 在 iPhone Safari 中刷新；若仍为旧版，彻底关闭桌面 App 后重开，必要时删除旧图标后重新添加到主屏幕。
