# 茶样志 PWA V1.2 正式升级版

本版本以可正常使用的 V1.1 完整版为基础升级，保留：

- 首页和统计
- 新建品评批次
- 自定义茶类
- 拍照或从相册选择照片
- OCR 包装文字识别
- 我的香气、滋味
- 他人的香气、滋味
- 快捷术语
- 本地保存
- 历史搜索
- CSV 导出
- JSON 备份和恢复

## V1.2 新增

1. 每个样品卡分别显示“核心香气”和“核心滋味”。
2. 系统综合“我的品评”和“他人的品评”统计关键词频率。
3. 每个换行、分号或句号视为一条独立意见。
4. 有重复关键词时，显示频率最高的关键词，最多显示3个。
5. 没有重复关键词时，按意见内容凝练显示最多2个代表词。
6. 核心词与“我的品评”一致时显示绿色。
7. 核心词未出现在“我的品评”中时显示红色。
8. CSV 中新增“核心香气”和“核心滋味”两列。

## 覆盖更新 GitHub Pages

1. 解压 ZIP。
2. 进入 GitHub 仓库 `tea-sample-log`。
3. 点击 `Add file` → `Upload files`。
4. 上传本文件夹中的全部文件和 assets 文件夹，不要上传 ZIP 本身。
5. 点击 `Commit changes`。
6. 等待右侧 Deployments 变为绿色。
7. 在 iPhone Safari 打开网页并刷新。
8. 若仍显示旧版，完全关闭桌面 App 后重开；仍未更新时，删除桌面图标，再从 Safari 重新“添加到主屏幕”。

## 数据安全

覆盖网页文件不会主动删除原来的本地记录。正式更新前仍建议在“设置与数据”中导出一次 JSON 备份。
