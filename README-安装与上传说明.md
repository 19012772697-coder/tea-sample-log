# 茶样志 PWA V1.0

## 在 Windows 上传到 GitHub Pages

1. 解压本压缩包，确认能看到 `index.html`、`app.js`、`styles.css`、`manifest.webmanifest`、`sw.js` 和 `assets` 文件夹。
2. 打开你的 GitHub 仓库，进入 **Add file / 添加文件 → Upload files / 上传文件**。
3. 把解压后的全部文件和 `assets` 文件夹一起上传到仓库根目录，不要只上传压缩包。
4. 点击 **Commit changes / 提交更改**。
5. 进入仓库 **Settings / 设置 → Pages**。
6. 在 **Build and deployment** 中选择 **Deploy from a branch**。
7. Branch 选择 `main`，文件夹选择 `/ (root)`，点击 **Save**。
8. 等页面显示网站地址后，用 iPhone 的 Safari 打开该地址。

## 安装到 iPhone 桌面

1. 必须使用 Safari 打开网站。
2. 点击 Safari 底部的“分享”按钮。
3. 向下找到“添加到主屏幕”。
4. 名称保留“茶样志”，点击“添加”。

## 使用提示

- 数据和照片默认保存在当前 iPhone 的 Safari 网站数据中。
- 请定期进入“设置与数据”，点击“备份全部数据（JSON）”。
- 不要随意清理 Safari 网站数据，否则本地记录可能被删除。
- CSV 文件可用 Excel、WPS 或 Numbers 打开。
- OCR 需要联网加载识别组件；内部样品或包装反光时，请人工检查。
- iPhone 网页语音识别兼容性有限，填写文本时可直接使用系统键盘上的麦克风听写。
